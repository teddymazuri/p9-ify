Privacy Policy
Last Updated: January 2026

P9-ify is committed to protecting your privacy. This policy explains how we handle your data.

1. Data Collection

No Data Collection: P9-ify does not collect, store, or transmit any personal or business data. All information you enter (employee details, payroll data, company information) remains exclusively in your browser's local storage.

Local Storage: Your data is stored locally on your device using your browser's local storage. We have no access to this data.

2. No Tracking
  • No cookies for tracking purposes
  • No analytics tracking
  • No user behavior monitoring
  • No data sharing with third parties
3. Your Control

You have complete control over your data:

  • Export: You can export all your data at any time
  • Delete: You can delete all data using the "Reset All Data" button
  • Backup: You are responsible for backing up your exported data
4. Browser Requirements

P9-ify requires modern browser features including:

  • Local Storage API for data persistence
  • JavaScript enabled
  • Modern CSS support
Important: Since data is stored locally, clearing your browser cache will delete your P9-ify data unless you've exported it first.

Data Security
Security First Approach

P9-ify is designed with privacy and security as core principles.

1. Local-Only Architecture
Client-Side Processing

All data processing occurs in your browser. No information is sent to any server.

No Server Communication:

  • No API calls with your data
  • No cloud storage
  • No external data processing

Complete Privacy:

  • Your data never leaves your device
  • No network transmission of sensitive info
  • Works completely offline
2. Browser Security Features

P9-ify leverages your browser's built-in security:

Local Storage

Data is stored using your browser's local storage, protected by same-origin policy.

Origin Isolation

Data is isolated to P9-ify's domain and cannot be accessed by other websites.

3. Data Protection Recommendations
Best Practices for Users
  • Export Regularly: Download backups of your data frequently
  • Secure Device: Use on trusted, password-protected devices
  • Clear Browser Data: Be cautious when clearing browser cache
  • Private Browsing: Avoid using incognito/private modes for long-term data
4. Security Limitations
Important Considerations

While P9-ify provides local security, you should be aware of:

  • Browser vulnerabilities (keep your browser updated)
  • Device security (use antivirus and firewalls)
  • Physical access to your device
  • Browser extensions that might access local storage
5. GDPR & Data Protection

P9-ify inherently complies with data protection principles:

  • Data Minimization: Only store what you enter
  • Purpose Limitation: Data used only for intended payroll functions
  • Storage Limitation: You control data retention
  • Integrity & Confidentiality: Local storage provides inherent protection
